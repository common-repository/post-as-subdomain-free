Convert post into Subdomain.

== Description ==


As the title WordPress Post as Subdomain Lite says it all this plugin convert any of your single Post into sub domain used for wordpress seo.

WordPress Post as Subdomain Pro empower WordPress to easily convert any single Post into Subdomains for website.

= IMPORTANT =


Do not forget to setup a WildCard Subdomain.

= Usage =


[youtube https://www.youtube.com/watch?v=G_3ySITP4O8]

Simply Create a Post, keep the Post id in mind. Then Visit Posts > Subdomains > Enter Post ID and hit save.
You may require one time Configuration of adding Wildcard Subdomain but most of the Host already did it. 
If you want to know how to add Wildcard Subdomain 
[Click here](https://codex.wordpress.org/Configuring_Wildcard_Subdomains/)

== Installation ==

1. Unzip `zip` and Upload Folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Posts > Subdomain > Enter Post ID > Save.


== Frequently Asked Questions ==

= How to configure a WildCard Subdomain =

Read the following tutorial https://codex.wordpress.org/Configuring_Wildcard_Subdomains


= Where to set the Post id to be subdomain? =

In Admin Dashboard Visit Post > Subdomain > Enter Post ID > Save

= Where to find Post ID? =

In your Admin Dashboard Click EDIT on any Post, and see the URL in address bar and file the value of post=YOURPostID
